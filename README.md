# Crossplane Provider for Cloud Foundry (Fork)

This is a fork of [SAP/crossplane-provider-cloudfoundry](https://github.com/SAP/crossplane-provider-cloudfoundry).

## Changes from upstream

- **Crossplane v2 support** — migrated to `crossplane-runtime/v2` with all CRDs namespace-scoped
- **Custom IDP origin** — support for custom Identity Provider origin in CF UAA password grant
